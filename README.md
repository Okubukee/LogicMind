## ✨ Características

### 🎯 **Múltiples Categorías de Algoritmos**
- 📊 **Ordenamiento** - Visualiza cómo los algoritmos organizan datos
- 🗺️ **Pathfinding** - Encuentra el camino más corto en un grid
- 🔍 **Búsqueda** - Explora diferentes técnicas de búsqueda
- 🔗 **Grafos** - Recorridos y análisis de estructuras de grafos

### 🎨 **Visualización Interactiva**
- Animaciones paso a paso
- Control de velocidad (0.25x a 5x)
- Controles de reproducción (pausa, paso a paso)
- Diseño moderno con tema oscuro y acentos verdes neón

### 🔧 **Tecnologías Modernas**
- Backend en **Python/Flask** con API REST
- Frontend en **React** con diseño responsive
- **TailwindCSS** para estilos modernos
- **Docker** para despliegue sencillo
